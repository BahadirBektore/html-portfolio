# HTML Portfolio

**HTML Portfolio** is a personal portfolio web page built using fundamental HTML and CSS.  
It is designed to showcase projects, skills, and contact information in a clean and structured layout.

---

## 🌐 Live Demo

Try it here:  
👉 https://bahadirbektore.github.io/html-portfolio/

---

## 🚀 Project Overview

This project includes:

- A clean and organized portfolio layout  
- Sections for About Me, Projects, Skills, and Contact  
- Navigation menu for easy browsing  
- Professional and responsive page structure

---

## 🛠 Technologies Used

This project was built using:

- **HTML5** – Semantic markup and page structure  
- **CSS3** – Styling, layout, and visual presentation

---

## 📋 How to Use

1. Open the live demo link above.  
2. Scroll through the sections to view projects and information.  
3. Resize your browser to see responsive behavior.
